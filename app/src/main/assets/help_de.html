Erste Schritte
Gehen Sie zu Einstellungen und wählen Ihre Sprache im Sprachmenü aus.

Weitere Optionen in den Einstellungen
Synchronisieren per E-Mail ist optional, aber eine empfohlene Funktion, die Ihre App mit einem kostenlosen Mitgliedschaftskonto auf Reverd.com synchronisiert.

Das Blockieren von Anrufen ohne Anruf-ID ist standardmäßig AN. Schalten Sie die Funktion aus, wenn Sie Anrufe ohne Anruf-ID erhalten möchten, die nicht auf der Blacklist stehen.

Sie können Anrufe aus bestimmten Gegenden oder von anderen Ländern blockieren, indem Sie diese auswählen.

Fügen der Blacklist eine Telefonnummer manuell hinzu
Tippen Sie auf das <84>+<93>-Symbol.
Füllen Sie das Formular aus und tippen Sie auf den Hinzufügen-Button.
Melden Sie die Telefonnummer in der Community-Blacklist (empfohlen).

Verschieben Sie Telefonnummer zwischen Ihren Listen
Tippen Sie auf das Symbol <84>Bearbeiten<93>.
Wählen Sie die Telefonnummern aus, die verschoben werden sollen.
Tippen Sie auf das Listensymbol, um die Nummer sofort zu verschieben.

Wenn Sie Anrufe von einer Nummer auf der Blacklist erhalten möchten, verschieben Sie diese einfach auf die Whitelist oder umgekehrt.

HINWEI:Das Verschieben von Nummern zwischen Listen beeinflusst nur Einträge auf Ihrem Telefon. Diese Funktion ändert den Status auf der Community-Blacklist nicht.

Automatische Updates für besseren Schutz (empfohlen)
Obwohl unsere kostenlose Anwendung voll funktionsfähig ist, empfehlen wir, dass Sie unsere Updates der Community-Blacklist abonnieren. Unsere Updates beinhalten die neuesten Telefonnummer, die als belästigend gelten und stellen ihn einen besseren Schutz gegen Spam zur Verfügung.
